On the previous part of this tutorial, the Java Service Wrapper has been configured. Now, it is time to test run the application and install the Windows Service.

Part 3 - Install the Application as a Windows Service

Steps:  

Test Run the Application

  • Open the Command Prompt (Note: Run it as Administrator).
  • Change the working directory to the bin folder
  • Type run

 

After getting a positive result, the Java Console Application can now be installed as a Windows Service and here's how:

  • While the application is running on the Command Prompt, press Ctrl + C. This will terminate the application from running.
  • Now type install-service and hit Enter.
  • This will install the Java Console Application as a Windows Service on the machine.
  • To run the Windows Service, simply type start-service and hit Enter.
  • To terminate the Windows Service, type stop-service and hit Enter.
  • To uninstall the Windows Service, type uninstall-service and hit Enter.

Now that the Java Console Application is installed as a Windows Service, it will always run when the system boots since it is configured to AUTO_START.
