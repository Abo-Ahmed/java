How to Install AlwaysUp

Start using our award-winning service wrapper in less than 1 minute!

AlwaysUp, our popular utility designed to run any application as a Windows Service, is very easy to install. Completing the standard steps below should only take you a few seconds...

To install AlwaysUp:

  1. Download AlwaysUp, if necessary:

    • Click here to get the latest version of AlwaysUp

    • Your browser may ask you to confirm the download. Choose to Run the installer if possible; otherwise save it to your hard drive and start it manually from there.

      Here is what you may see when downloading with Internet Explorer (but without the cool looking boats):

  2. Double-click on the installer (AlwaysUp_Installer.exe) to start it. This will not be necessary if you downloaded and selected Run in the previous step.

  3. You may see the following security prompt on Windows 7, Vista and Server 2008 if you have User Account Control (UAC) enabled:

    AlwaysUp is digitally signed for your protection and our company name (Core Technologies Consulting, LLC) should be prominently displayed.

    Click Yes to continue.

  4. The installation starts off with a welcome screen. Click Next > to continue:

  5. You have arrived at the obligatory license agreement screen! Select the I accept the agreement option and click Next > when you are ready to continue:

  6. Enter your name and organization (optional) and click Next > to move on:

  7. By default, AlwaysUp will be installed in C:\Program Files\AlwaysUp (or C:\Program Files (x86)\AlwaysUp on 64-bit systems). If you wish to install it elsewhere, enter the new location and click Next > to continue:

  8. AlwaysUp creates an entry in your Start menu and you can alter its name here if you wish. Click Next > to proceed:

  9. Choose the icons that will make it convenient to run AlwaysUp. As usual, click Next > to continue:

  10. We're ready to install! Click Install to start the magic:

  11. It will take a few seconds to copy the files:

  12. This is the final step. Select what you would like to do next (start AlwaysUp, watch a how-to video, etc.) and click Finish.

  13. Congratulations, you're all done. Go grab some chocolate to celebrate!

    Next steps:

    • Watch a short video showing how to set up the Notepad application as a service with AlwaysUp
    • Browse our step-by-step tutorials to configure popular applications with AlwaysUp (including Dropbox, Firefox, Java, Batch files, Wireshark, and many more)

Trouble installing AlwaysUp?

  • Consult the AlwaysUp FAQ for answers to commonly asked questions and troubleshooting tips.
  • Contact us and we will be happy to help!
